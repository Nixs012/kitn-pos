## 📡 System Requirements
- **Browser**: Google Chrome, Microsoft Edge, or Safari (latest versions).
- **Connectivity**: Real-time features require internet. Offline mode supports limited cart operations.
- **Scanner**: Compatible with both handheld USB/HID scanners and mobile cameras.
